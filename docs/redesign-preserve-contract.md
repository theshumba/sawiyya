**Recognizer / engine fixes already shipped (`src/recognizer/knn.ts` + CameraTrainer) ‚Äî NEVER touch logic:**

**Design tokens that MUST exist (`tailwind.config.js`) ‚Äî primitives in `ui.tsx` depend on them:**
