Rule of thumb: design the Arabic screen first, not last. If a layout only works in English and breaks when mirrored — crowded action rows, left-anchored numbers, icons that imply direction — it's the layout that's wrong, not the translation. Every screen in this set was checked in both directions before it shipped.
